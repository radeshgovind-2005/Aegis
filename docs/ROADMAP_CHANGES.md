# Roadmap Changes

**Do NOT silently edit `ROADMAP.md`. Always propose here first.**

Format per proposal:

```
## YYYY-MM-DD — Short title
**Proposed by:** Claude | <human name>
**What:** Specific change to the roadmap.
**Why:** What made this necessary.
**Impact:** Which phases affected.
**Status:** proposed | approved | rejected | implemented
**Approval:** (human fills in)
```
